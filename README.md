# Car Price Prediction

## Overview
This project predicts the selling price of used cars using Machine Learning. A Linear Regression model is trained on historical car data to estimate the selling price based on various features.

## Features
- Data Preprocessing
- Feature Engineering
- Linear Regression Model
- Model Evaluation
- Car Price Prediction
- Data Visualization

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset Features
- Year
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner

## Model Performance
- Training R² Score: **0.8835**
- Testing R² Score: **0.8401**

## Output
The model predicts the selling price of a car based on the given input features.
